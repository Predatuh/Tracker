🔄 Over-the-Air EXE Update

Upload a new LBDTracker.exe here. Every time the EXE is launched it checks this server and automatically downloads & installs the new version.

How it works
1. Build the new EXE with PyInstaller (bump version.txt first)
2. Upload it here
3. Anyone running the old EXE will be updated automatically next time they open it
4. Make sure update_url in their config.ini points to this site
