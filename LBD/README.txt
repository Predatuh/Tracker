LBD TRACKER - PDF to LBD Tracker Application
=============================================

WHAT IS THIS?
This is a standalone application for tracking LBD (Large Block Defects) from PDF documents.

HOW TO USE:
1. Double-click LBDTracker.exe to launch the application
2. Upload a PDF file using the "Upload PDF" section
3. Extract pages from the PDF
4. Create and manage LBD trackers for your project
5. Track status with color-coded labels

FEATURES:
- PDF page extraction and image processing
- Power block management
- LBD tracking with 6 status types
- Site map visualization
- Real-time status updates

REQUIREMENTS:
- Windows 7 or later
- No installation required
- No additional software needed

NOTES:
- All data is stored locally on your computer
- The application creates a database on first run
- Database is stored in the same folder as the EXE
